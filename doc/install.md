# Install Mia
Back to [README](../README.md)

Mia can be compiled and run on multiple platforms!

Mia is cmake based and can also be setup for IDE's to debug the code.

Coding works fine on most platforms, but I like MacOs (with my MacAir M3) the most.
Cause: Crazy fast to iterate + Google sanitizers (The latter works on MacOs and Linux (Ubuntu)). 

Have a look at the following options to install all needed libraries to compile and run Mia.

## Platforms to compile Mia
- [MacOs](./install_macos.md)
- [Windows](./install_windows.md)
- [Linux](./install_linux.md)
- [Android](./install_android.md)
